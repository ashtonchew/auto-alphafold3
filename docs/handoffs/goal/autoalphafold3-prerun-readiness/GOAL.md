# AutoAlphaFold3 Pre-Run Readiness Goal

Implement the pre-run readiness layer for the NanoFold-style AlphaFold3-lite system before autonomous search starts.

This goal covers structured pre-registration, Falsification Gate contracts and pure verdict logic, baseline-readiness validation, confirmed-only Discovery Ledger helpers, two-stage orchestration, a fakeable Modal gate-wave adapter, and readiness CLI/reporting.

Autonomous research search is out of scope. Fake benchmark results, fake Arrow files, fake Modal runs, fake baseline metrics, fake gate verdicts, fake discovery records, and fake validation labels are forbidden.
